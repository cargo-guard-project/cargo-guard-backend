# CargoGuard Web

React/Vite web application for CargoGuard lab work 3.

## Setup

```powershell
Copy-Item .env.example .env
npm install
npm run dev
```

The default API URL is:

```text
VITE_API_BASE_URL=http://localhost:3000
```

Run the backend from the repository root before using the web application.

## Build

```powershell
npm run build
```
